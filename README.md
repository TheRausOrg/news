# News site (news.theraus.org)

This repo publishes the **news/prayer** site for TheRausOrg.

## Locale paths
- Shared German: /de/
- German (Switzerland): /de-CH/
- German (Germany): /de-DE/
- Shared English: /en/
- English (UK): /en-GB/
- English (US): /en-US/

## Language chooser
- / (root) shows a warm chooser and stores the preference cookie: 
  - name: `lang`
  - value: one of the locale paths above

## Edit content
Start with:
- `de/index.html`
- `en/index.html`

Use region variants only when content differs.
